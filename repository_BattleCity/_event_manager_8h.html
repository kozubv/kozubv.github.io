enum EventType
strong

типы событий пользовательского управления

KeyDown-TextEntered - стандартное перечисление типов библиотеки SFML для единого образа обработки событий sf::Event и "Событий реального времени" классов sf::Keyboard, sf::Mouse и sf::Joystick. Преобразование к единой форме идентификации событий позволяет однообразно настраивать кнопки пользовательского ввода в текстовом файле (EventManager::LoadBindings)
