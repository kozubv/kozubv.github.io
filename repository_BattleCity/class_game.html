Detailed Description

запускает бесконечный цикл обновления и отображения игры.

Управление режимами программы (главное меню, битва, статистика и переключение уровней) построено на базе паттерна "Состояние". Класс Game является реализацией компонента "Клиент" этого паттерна. В реализации экземпляр Game обращается к экземпляру StateManager как к абстрактному режиму, не зависимо от правил последовательности переключения и текущих режимов. Для обмена данными между режимами (конкретными состояниями) Game агрегирует и экземпляр SharedContext.
