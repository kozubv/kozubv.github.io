Detailed Description

общий интерфейс режимов программы. компонент "базовый интерфейс состояний" паттерна "Состояние".

Для обеспечения наличия независимых режимов в программе, возможности их отдельной разработки и комбинирования в порядке взаимодействия применен паттерн State.
