Detailed Description

элемент "Leaf" паттерна "Компоновщик" для визуальных элементов

Обладает правилами позиционирования аналогичными VisualPanel. VisualPanel Управляет набором спрайтов, при этом не является их владельцем. т.е. за создание и удаление не отвечает.
