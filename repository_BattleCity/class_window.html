Detailed Description

Абстракция графического окна ОС.

Отделяет конкретную используемую библиотеку (SFML в данном случае) от использования функционала окна ОС в коде программы. Обеспечивает : разворачивание, закрытие, отслеживание фокуса мыши, начало и конец отображения графики, обновление программы и изображения, обработку событий пользовательского ввода при помощи EventManager.
