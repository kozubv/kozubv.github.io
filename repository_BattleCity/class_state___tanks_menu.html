Detailed Description

Режим главного меню. Выбор битвы или выход из программы.

Режим программы по умолчанию. Дает возможность перейти к выбору уровня или выйти из программы. Отображает логотип, курсор и пункты меню.

Note
перед использованием режим должен быть зарегистрирован при помощи StateManager::RegisterState
